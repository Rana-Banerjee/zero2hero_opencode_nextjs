Do the following steps in order:

1. Run /compact to summarise the current conversation

2. Update CONTEXT.md with the current project state. Replace the entire file
   with a fresh snapshot covering:
   - Status: current milestone and what you are working on
   - What's Done: any new items completed this session
   - What's Next: the immediate next 2-3 tasks
   - Active Decisions: any mid-session decisions not yet in PROJECT.md
   - Files Most Recently Touched: list the files changed this session

3. Confirm with: "Session wrapped. CONTEXT.md updated. Safe to exit."

4. Run /exit
